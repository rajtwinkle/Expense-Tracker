# Expense Tracker

A responsive web-based Expense Tracker application that helps users manage and monitor their daily expenses. Users can add, view, and delete expenses while tracking their spending efficiently through an intuitive interface.

## Features

- Add new expenses with title, amount, category, and date
- View all recorded expenses
- Delete unwanted expense records
- Automatically calculate total expenses
- Responsive and user-friendly interface
- Data persistence using Local Storage
- Real-time expense tracking

## Technologies Used

- HTML5
- CSS3
- JavaScript
